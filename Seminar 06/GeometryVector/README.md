# 📐 GeometryVector 

Да се реализира клас, който представя геометричен вектор с **произволна** размерност.

Да бъде имплементирано:

- създаване на вектор с подадена размерност

- достъп за четене/модификация на координатите на дадена позиция, чрез оператор`[]`

- проверка дали векторът е нулев чрез conversion оператор към `bool`

- извеждане и въвеждане чрез потоковите оператори за форматиран вход и изход `>>` и `<<`

- унарен оператор `+`
- унарен оператор `-`, който връща противоположния вектор

- събиране на два вектора (`+`, `+=`)
- изваждане на два вектора (`-`, `-=`)
- умножение на вектор със скалар (`*`, `*=`)
- деление на вектор със скалар (`/`, `/=`)

- скаларно произведение между два вектора

- проверка дали два вектора са ортогонални

- проверка за еквивалентност на два вектора (`==`, `!=`)

- сравнение на вектори

При сравнението на два вектора първо може да се сравнява размерността им, а при равна размерност — координатите им лексикографски. 

>Тъй като координатите са от тип double, сравнението чрез `<=>` трябва да връща подходящ comparison category за floating-point стойности, тоест `std::partial_ordering`.

>Операциите между два вектора са допустими **само** ако те са с **една и съща размерност**.
---

#### Пример за работа с класа (виж `main.cpp`):

Вход (два вектора с по 3 координати):
```
1 1 1
1 2 3
```

```
v1 = (1, 1, 1)
v2 = (1, 2, 3)
v1[0] = 1
+v1 = (1, 1, 1)
-v1 = (-1, -1, -1)
v1 + v2 = (2, 3, 4)
v1 - v2 = (0, -1, -2)
v1 * 2 = (2, 2, 2)
3 * v2 = (3, 6, 9)
v1 / 2 = (0.5, 0.5, 0.5)
Scalar product: 6
Orthogonal: 0
v1 == v2: 0
v1 != v2: 1
v1 < v2: 1
v1 > v2: 0
v1 <= v2: 1
v1 >= v2: 0
v1 is not the zero vector
v3 = (1, 1, 1)
After v3 += v2: (2, 3, 4)
After v3 -= v1: (1, 2, 3)
After v3 *= 2: (2, 4, 6)
After v3 /= 2: (1, 2, 3)
```
